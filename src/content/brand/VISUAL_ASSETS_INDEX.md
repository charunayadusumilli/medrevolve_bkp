## Typography

**Headings:** Serif font (e.g., Georgia, Merriweather)
- Use for: Main headlines, section titles, quotes

**Body text:** Sans-serif font (e.g., Inter, Helvetica Neue)
- Use for: Paragraph text, captions, UI labels

**Special use:** Monospace for compliance card labels

## File naming convention

All assets follow this pattern:

`[platform]-[type]-[size].[ext]`

Examples:
- `facebook-cover-1640x856.png`
- `linkedin-banner-1500x500.png`
- `instagram-hero-1080x1080.png`
- `youtube-thumbnail-1280x720.png`
