# MedRevolve · Meta Business Suite Upload Checklist

> 25 minutes, two pages, every field. Open `COPY_BUNDLE.md` and work top to bottom.

---

## Before you start

Open `COPY_BUNDLE.md` in any text editor — all the copy you'll paste lives there, ready to drop. Log in at `business.facebook.com`.

---

## Part A — Facebook Page (10 minutes)

### A1 · Profile picture
- Go to your MedRevolve page → Edit profile photo
- Upload `png/facebook/profile-320x320.png`
- Crop: leave alone (already 1:1, ready for the circular mask)

### A2 · Cover photo
- Same page → Edit cover photo → Upload photo
- Upload `png/facebook/cover-1640x856.png`
- **Important:** Facebook will preview mobile vs desktop. The mobile-safe area is the center 820 × 312, which is where the "Twenty / founding / operators." headline sits. Compliance ledger card (left) and stats (right) are safe to be cropped on mobile.

### A3 · Page name
- Settings → Page info → Page name
- Enter: `MedRevolve`

### A4 · Username (vanity URL)
- Settings → Page info → Username
- Enter: `@medrevolveplatform` (or `@medrevolveB2B` if first is taken)

### A5 · Category
- Settings → Page info → Category
- Select: *Software*, *Healthcare service*, *B2B service*

### A6 · About section
- Settings → Page info → About
- Paste from COPY_BUNDLE.md § 01 "About / company overview (long)"

### A7 · Website
- Settings → Page info → Website
- Enter: `medrevolve.com/forbusiness`

### A8 · Short description
- Page → Edit → Short description field (under cover photo)
- Paste from COPY_BUNDLE.md § 01 "Short description (255 chars)"
- Character limit: 255

## Verification Checklist

Before you publish any posts:

- [ ] All profile pictures are consistent across platforms
- [ ] All bios/headlines direct to `medrevolve.com/forbusiness`
- [ ] No consumer language in any bio (we're B2B)
- [ ] Compliance messaging is accurate (refer to brand docs)
- [ ] Website links all work and point to correct pages
- [ ] Category selections match our positioning (B2B, not consumer)
- [ ] Cover photos are cropped correctly for mobile vs desktop
- [ ] All copy is spell-checked and on-brand

## Post-Launch

After setup:
1. **Pin one introductory post** on Facebook and LinkedIn explaining what MedRevolve is
2. **Turn on messaging** on Instagram and Facebook (route to support email)
3. **Set up Meta Business Suite** to manage both Facebook and Instagram from one dashboard
4. **Enable notifications** for comments and messages
5. **Archive, don't delete**, old posts from previous branding (if any)
