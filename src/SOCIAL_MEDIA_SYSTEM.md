## 🚀 How to Use

### Option 1: Manual Posting (Recommended for Start)
1. Go to `/SocialMediaDashboard`
2. Click "Create Post" tab
3. Use **AI Content Generator** to create compliant caption
4. Add image URL (Unsplash, your CDN, or Google Drive)
5. Review and customize caption
6. Click "Publish to Instagram"

### Option 2: Semi-Automated
1. Generate multiple captions using AI
2. Save them to a document
3. Schedule time weekly to post manually
4. Mix telehealth, business, and compliance content

### Option 3: Full Automation
1. Go to "Auto-Post" tab
2. Enable auto-posting
3. Set frequency (daily/weekly)
4. System generates and posts automatically
5. Review posted content in "Overview"

## 🔧 Technical Details

### Backend Functions
1. **`generateSocialMediaContent`**
   - Generates compliant captions
   - Provides image prompts
   - Includes hashtags & compliance notes

2. **`createSocialPost`**
   - Posts to Instagram/Facebook
   - Saves to SocialPost entity
   - Error handling & validation

3. **`checkInstagramConnection`**
   - Verifies OAuth status
   - Returns user info
   - Connection health check

4. **`bulkPostToSocial`**
   - Posts multiple items at once
   - Batch processing
   - Success/failure tracking

### Database Entity
**SocialPost** entity tracks:
- Platform (Instagram/Facebook)
- Caption text
- Image URL
- Post ID from platform
- Status (draft/scheduled/published/failed)
- Engagement metrics
- Topic category
- Content type
